# 基于Vue 2.0 + Vuex + ElementUI 的开发模板


## 构建项目

```bash
npm run start           # 开发环境自动编译并开启mock服务
npm run build           # 打包到static目录,同时在上级文件夹中生成index.cshtml
```

## 说明
element-variables.css是ElementUI的样式配置文件，需要自己打包修改
```bash
et -o ./src/theme/
```
字体图标用的阿里的图标库 ![iconfont](http://www.iconfont.cn/)
